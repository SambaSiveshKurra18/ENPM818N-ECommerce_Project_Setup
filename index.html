ENPM818N-E_Commerce_Application_Setup

Prerequisites

  • Ubuntu Server (EC2 instance)
  • Apache2 installed
  • PHP and MySQL installed
  • MySQL Workbench (optional)

Step-by-Step Instructions

1. Update the EC2 instance

sudo apt update

2. Install Apache2 Web Server

sudo apt install apache2

3. Check UFW Firewall Application Profiles

sudo ufw app list

4. Start the Apache2 service

sudo systemctl start apache2

5. Get the Public IP of your EC2 Instance

curl http://icanhazip.com

6. Verify Apache2 Installation

Go to your browser and navigate to: http://<your-ec2-public-ip>

7. Install PHP and Required Modules

sudo apt install php libapache2-mod-php php-mysql

8. Check PHP Version

php -v

9. Configure Apache to Prioritize index.php

Open the dir.conf file:

sudo nano /etc/apache2/mods-enabled/dir.conf

Change from:

<IfModule mod_dir.c>
    DirectoryIndex index.html index.cgi index.pl index.php index.xhtml index.htm
</IfModule>

To:

<IfModule mod_dir.c>
    DirectoryIndex index.php index.html index.cgi index.pl index.xhtml index.htm
</IfModule>

10. Restart Apache2 Service

sudo systemctl restart apache2

11. Check Apache2 Status

sudo systemctl status apache2

12. Remove Default index.html

cd /var/www/html
sudo rm -rf index.html

13. Move Project Files to Web Root

sudo mv * /var/www/html

14. Configure Database Connection

Navigate to the includes folder and open the connect.php file:

cd /var/www/html/includes
sudo nano connect.php

Update with your RDS database connection details and save.

15. Test the Application

In your browser, navigate to: http://<your-ec2-public-ip>/index.php

Configuring the Database

1. Create a Database

Login to MySQL Workbench or via CLI and create the required database.

2. Execute the SQL Script

Find the ecommerce_1.sql file in the database folder, open it in MySQL Workbench, and execute the queries.

3. Reload the Application

Once the tables are created, reload the application by visiting:

http://<your-ec2-public-ip>/index.php
